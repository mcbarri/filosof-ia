Política de Privacidad

Filosof-IA — Aplicación de publicación de contenido educativo.

Última actualización: 27 de mayo de 2026

Información que recopilamos

Esta aplicación no recopila, almacena ni comparte información personal de usuarios. Solo accede a los datos necesarios para publicar contenido en la página de Facebook Filosof-IA.

Uso de datos

La aplicación utiliza la API de Facebook exclusivamente para:

  • Publicar videos y contenido en la página Filosof-IA
  • Leer métricas públicas de interacción (likes, comentarios, vistas)

No se comparte información con terceros.

Seguridad

La aplicación opera bajo estrictas medidas de seguridad proporcionadas por la plataforma de Facebook. No se almacenan datos de usuarios en servidores externos.

Contacto

Para cualquier pregunta sobre esta política de privacidad, contactar a través de la página de Facebook Filosof-IA.
