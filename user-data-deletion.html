Eliminación de Datos de Usuario

Última actualización: 27 de mayo de 2026

¿Qué datos almacenamos?

Filosof-IA no recopila ni almacena datos personales de usuarios. Nuestra aplicación únicamente publica contenido en la página de Facebook Filosof-IA utilizando la API de Facebook para páginas.

Solicitar eliminación de datos

Si has interactuado con la aplicación y deseas que tus datos sean eliminados, puedes enviar una solicitud a nuestro correo de soporte. Procesaremos tu solicitud dentro de 48 horas hábiles.

Para solicitar la eliminación de tus datos, envía un correo a:

mcbarri.gt@gmail.com

Incluye en el asunto: "Solicitud de Eliminación de Datos - Filosof-IA"

Proceso de eliminación

  1. Recibimos tu solicitud por correo electrónico
  2. Verificamos tu identidad
  3. Eliminamos cualquier dato asociado (si existe)
  4. Te confirmamos la eliminación por correo

Sin cuenta de usuario

Filosof-IA no requiere registro de usuarios ni crea cuentas. La aplicación solo interactúa con la página de Facebook. Por lo tanto, no hay cuentas de usuario que eliminar.
